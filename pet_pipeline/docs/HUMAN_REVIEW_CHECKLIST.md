# Human Review Checklist

Review every candidate before promoting it to `output/final/`.

## Identity checks

- Eye color matches the character spec.
- Fur pattern matches the character spec.
- Ears match the character spec.
- Tail length and thickness look stable.
- Nose and mouth stay centered and natural.

## Expression checks

- The requested state is clearly visible.
- The emotion is not so strong that the character identity breaks.
- The eyes and mouth still look like the same cat.

## Composition checks

- Canvas size is correct.
- Character is centered consistently.
- Body proportions match the approved baseline.
- All paws are visible unless intentionally hidden.

## Asset checks

- Background has been removed if this asset is final.
- Transparent PNG renders cleanly without halos.
- File name follows `{character}_{state}_{stage}_v{n}.png`.
- Metadata and manifest entries are present.
